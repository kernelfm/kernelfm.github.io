LostCases

Плагин для управления утерянными кейсами и автоматического контроля их статуса.

Установка

Поместите файл плагина в директорию /plugins и перезапустите сервер.

Конфигурация

Настройки находятся в файле config.yml. Основные параметры:

  • check-interval: интервал проверки в секундах
  • auto-save: автоматическое сохранение данных

Команды

  • /lc help — показать список доступных команд
  • /lc reload — перезагрузить конфигурацию плагина
  • /lc status — проверить текущее состояние работы
