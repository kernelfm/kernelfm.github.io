1. Структура конфигурации кейса (cases/*.yml) [cite: 3]

Плагин поддерживает неограниченное число кейсов. [cite: 4] Конфигурация каждого кейса располагается в папке plugins/LostCases/cases/ в формате YAML. [cite: 4]

  • display-name - Отображаемое название кейса. Поддерживает форматирование HEX-цветов в виде &#RRGGBB [cite: 5].
  • animation - Системный идентификатор анимации (например: chests) [cite: 6].
  • gui - Параметры меню открытия. Настраиваются размер (rows от 1 до 6), заголовок и предметы в слотах [cite: 7].
  • hologram - Текст голограммы DecentHolograms над блоком кейса в мире, высота смещения (y-offset) и список строк [cite: 8].
  • rewards - Список наград, их шансы и действия. Действие может содержать префиксы [command], [broadcast], [message] [cite: 9].

2. Создание аддона с новой анимацией (по шагам) [cite: 10]

Аддоны представляют собой обычные Java-библиотеки (Jar-файлы), которые помещаются в папку plugins/LostCases/addons/ и динамически загружаются плагином при старте. [cite: 11] Никаких дополнительных манифестов и plugin.yml файлов внутри Jar аддона создавать НЕ нужно. [cite: 12] Загрузчик плагина сам сканирует все скомпилированные классы в архиве. [cite: 13]

Шаг 1. Настройка Maven или Gradle зависимостей [cite: 14]

Вам необходимо подключить в качестве зависимостей (compileOnly / provided) скомпилированный Jar плагина LostCases, Paper API вашей версии (1.21.1) и DecentHolograms API. [cite: 15]

Шаг 2. Создание главного класса аддона [cite: 16]

Главный класс аддона должен реализовывать интерфейс ru.lostdev.lostCases.api.CaseAddon. [cite: 17] Плагин найдет его автоматически при сканировании Jar-файла. [cite: 17]

package ru.lostdev.myaddon;

import ru.lostdev.lostCases.LostCases;
import ru.lostdev.lostCases.api.CaseAddon;

public class MyAddonMain implements CaseAddon {
    private LostCases plugin;

    @Override
    public void onLoad(LostCases plugin) {
        this.plugin = plugin;
        plugin.getAnimationManager().registerAnimation(new MyCustomAnimation(plugin));
        plugin.getLogger().info("Аддон с кастомной анимацией успешно загружен!");
    }

    @Override
    public void onUnload() {
        plugin.getLogger().info("Аддон выгружен.");
    }
} [cite: 18]

Шаг 3. Имплементация анимации и сессии открытия [cite: 19]

Для написания анимации необходимо имплементировать интерфейс CaseAnimation, который будет возвращать название вашей анимации и создавать новую сессию открытия при взаимодействии игрока с кейсом. [cite: 20]

package ru.lostdev.myaddon;

import eu.decentsoftware.holograms.api.DHAPI;
import org.bukkit.Bukkit;
import org.bukkit.Location;
import org.bukkit.Material;
import org.bukkit.Sound;
import org.bukkit.block.Block;
import org.bukkit.block.Lidded;
import org.bukkit.configuration.ConfigurationSection;
import org.bukkit.entity.Player;
import org.bukkit.event.block.Action;
import org.bukkit.scheduler.BukkitRunnable;
import org.bukkit.scheduler.BukkitTask;
import ru.lostdev.lostCases.LostCases;
import ru.lostdev.lostCases.api.AnimationSession;
import ru.lostdev.lostCases.api.CaseAnimation;
import ru.lostdev.lostCases.case_system.CaseModel;
import ru.lostdev.lostCases.case_system.RewardModel;
import ru.lostdev.lostCases.utility.ColorUtil;
import java.util.Arrays;
import java.util.List;

public class MyCustomAnimation implements CaseAnimation {
    private final LostCases plugin;

    public MyCustomAnimation(LostCases plugin) {
        this.plugin = plugin;
    }

    @Override
    public String getName() {
        return "my_custom_anim";
    }

    @Override
    public AnimationSession createSession(Player player, Location location, CaseModel caseModel, ConfigurationSection config) {
        return new MyCustomSession(plugin, player, location, caseModel, config);
    }
}

class MyCustomSession implements AnimationSession {
    private final LostCases plugin;
    private final Player player;
    private final Location location;
    private final CaseModel caseModel;
    private final ConfigurationSection config;
    private Block spawnedBlock;
    private String holoName;
    private BukkitTask task;
    private boolean finished = false;

    public MyCustomSession(LostCases plugin, Player player, Location location, CaseModel caseModel, ConfigurationSection config) {
        this.plugin = plugin;
        this.player = player;
        this.location = location;
        this.caseModel = caseModel;
        this.config = config;
    }

    @Override
    public void start() {
        Location blockLoc = location.clone().add(0, 1, 0);
        spawnedBlock = blockLoc.getBlock();
        spawnedBlock.setType(Material.CHEST);
        plugin.getAnimationManager().registerBlock(blockLoc, this);
        location.getWorld().playSound(location, Sound.ENTITY_FIREWORK_ROCKET_LAUNCH, 1.0f, 1.0f);
        player.sendMessage("§eНажмите ЛКМ по сундуку для открытия!");
    }

    @Override
    public void handleInteract(Player clicker, Block block, Action action) {
        if (!clicker.getUniqueId().equals(player.getUniqueId()) || action != Action.LEFT_CLICK_BLOCK) {
            return;
        }
        if (finished) {
            return;
        }
        finished = true;
        if (block.getState() instanceof Lidded lidded) {
            lidded.open();
        }
        block.getWorld().playSound(block.getLocation(), Sound.BLOCK_CHEST_OPEN, 1.0f, 1.0f);
        RewardModel reward = caseModel.getRandomReward();
        if (reward != null) {
            plugin.getCaseManager().addHistoryEntry(caseModel.getName(), player.getName(), reward.getDisplayName(), reward.getMaterial());
            for (String cmd : reward.getCommands()) {
                String formatted = cmd.replace("%username%", player.getName()).replace("%group%", reward.getId());
                if (formatted.startsWith("[command] ")) {
                    Bukkit.dispatchCommand(Bukkit.getConsoleSender(), formatted.substring(10));
                } else if (formatted.startsWith("[broadcast] ")) {
                    Bukkit.broadcastMessage(ColorUtil.color(formatted.substring(12)));
                } else if (formatted.startsWith("[message] ")) {
                    player.sendMessage(ColorUtil.color(formatted.substring(10)));
                } else {
                    Bukkit.dispatchCommand(Bukkit.getConsoleSender(), formatted);
                }
            }
            if (Bukkit.getPluginManager().isPluginEnabled("DecentHolograms")) {
                holoName = "myaddon_" + player.getUniqueId() + "_" + System.currentTimeMillis();
                Location holoLoc = block.getLocation().clone().add(0.5, 1.5, 0.5);
                List lines = Arrays.asList(reward.getDisplayName().replace('&', '§'), "#ICON: " + reward.getMaterial());
                DHAPI.createHologram(holoName, holoLoc, false, lines);
            }
        }
        task = new BukkitRunnable() {
            @Override
            public void run() {
                stop();
            }
        }.runTaskLater(plugin, 80L);
    }

    @Override
    public void stop() {
        finished = true;
        if (task != null) {
            task.cancel();
        }
        if (holoName != null && Bukkit.getPluginManager().isPluginEnabled("DecentHolograms")) {
            DHAPI.removeHologram(holoName);
        }
        if (spawnedBlock != null) {
            plugin.getAnimationManager().unregisterBlock(spawnedBlock.getLocation());
            spawnedBlock.setType(Material.AIR);
        }
        plugin.getAnimationManager().endSession(player, location);
    }
} [cite: 21]

Шаг 4. Компиляция и развертывание аддона [cite: 22]

  • Скомпилируйте ваш проект Java в Jar-архив с помощью Maven (mvn clean package) или Gradle (shadowJar / build) [cite: 23].
  • Поместите полученную библиотеку Jar в каталог plugins/LostCases/addons/ вашего сервера [cite: 24].
  • Укажите в конфигурации cases/ имя вашей анимации: animation: my_custom_anim [cite: 25].
  • Введите команду /lostcases reload от имени администратора сервера [cite: 26]. Загрузчик плагина полностью обновит зависимости и зарегистрирует вашу анимацию без перезапуска всего сервера [cite: 26].
